日语文献 OCR 校读工具｜Windows 本地版
========================================

【怎么打开】

双击：开始使用（本地版）.bat

首次打开时，程序会自动准备运行环境并下载所需组件，可能需要几分钟。
以后再次双击会直接启动，并自动打开浏览器网页。


【使用期间】

请不要关闭启动后出现的黑色窗口。
网页地址固定为：http://127.0.0.1:7860/

127.0.0.1 代表“这台电脑自己”，同一局域网内的其他人无法访问。


【怎么关闭】

完成工作后，直接关闭黑色启动窗口即可停止工具。


【API Key 安全】

1. 本地版不会把 API Key 写入服务器文件，也不会经过 HF Space。
2. 默认不会长期保存 Key；关闭页面后需重新填写。
3. 如果这是自己的可信电脑，可勾选“在这台电脑的浏览器中记住 API Key”。
4. 随时可在 API 设置中点击“清除已保存密钥”。
5. OCR 文件仍会直接发送给百度 PaddleOCR API；翻译文本仍会直接发送给所选大模型 API。


【启动失败怎么办】

请不要立即关闭黑色窗口，把窗口中的报错截图发给维护者即可。

若提示没有 Python：
访问 https://www.python.org/downloads/windows/ 安装 Python 3.10 或更高版本，
安装时勾选“Add Python to PATH”，然后重新双击启动文件。

若首次安装时网络失败：
确认网络或代理软件可用，稍后重新双击即可，程序会自动继续。
