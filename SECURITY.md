# 安全说明

## 凭据

本项目不会把 API Key 持久化到本地后端磁盘。Key 在任务执行时从浏览器发送给本机后端，再用于请求对应的第三方 API。

浏览器中的“记住 API Key”默认关闭。启用后，Key 会保存在当前浏览器的 localStorage 中。请只在个人可信电脑上使用此选项。

## 网络边界

Windows 本地版默认仅监听 `127.0.0.1`。请不要在不了解风险的情况下改为 `0.0.0.0`，否则同一网络内其他设备可能访问页面和内存中的任务结果。

## 外部服务

- OCR 文件会发送给百度 PaddleOCR API；
- 待翻译文本会发送给用户选择的大模型 API；
- 本项目无法控制第三方服务的数据保留、地区可用性、费用或安全策略。

使用前请阅读相应服务商的隐私政策和服务条款。

## 报告安全问题

请勿在公开 Issue 中提供：

- PaddleOCR Token 或模型 API Key；
- 完整请求头；
- 未公开文献、私人批注或含个人信息的截图；
- 可直接利用的敏感数据。

报告时请提供最小化的复现步骤，并用占位符替换所有凭据。
