# 贡献指南

感谢你愿意帮助改进日语文献 OCR 校读工具。

## 报告问题

提交 Issue 时，请尽量包括：

- Windows、Python 与浏览器版本；
- 使用的是 PDF、图片、Markdown 还是校读存档；
- 问题出现前的操作步骤；
- 页面日志中的错误信息；
- 能否稳定复现。

请先删除截图、日志和示例文件中的 API Key、Token、真实姓名及未公开文献内容。

如果错误来自外部服务，请说明 HTTP 状态码和服务商，例如 PaddleOCR 500、模型 API 401 或 429，但不要粘贴完整请求头。

## 提交代码

1. Fork 仓库并创建独立分支；
2. 保持改动范围清晰，避免同时混入无关格式调整；
3. 在 Windows 本地版和浏览器中验证主要流程；
4. 确认没有提交 `.venv`、API Key、测试文献、校读存档或导出文件；
5. 在 Pull Request 中说明改动目的、验证方式和可能影响。

## 基本检查

提交前至少运行：

```bash
python -m py_compile server.py
```

如修改了 `开始使用（本地版）.bat` 或 `start-local.ps1`，请验证：

- 首次运行能创建 `.venv`；
- 再次运行不会重复安装依赖；
- 服务仅监听 `127.0.0.1`；
- 关闭启动窗口后服务停止。

## 安全问题

涉及凭据泄漏、任意文件访问或远程访问边界的问题，不要附带真实 Key 或敏感材料。公开披露前请先以私下方式联系维护者。
