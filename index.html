關於本工具

本工具集成了 OCR 文字識別與大語言模型翻譯，為學術文獻（首要面向日語，同時支持英、韓、中、法、德、拉丁等多語）提供快速的識別、翻譯與三欄對照校讀功能，是《新撰字鏡》數字人文工作平台的子系統。

工作流程
  1. 上傳 PDF 或掃描圖片（或直接導入已有 OCR 文本），並選擇源語言／目標語言
  2. PaddleOCR 進行版面分析與 OCR 識別（支持竪排、表格、多欄版式）
  3. 大語言模型（Claude / DeepSeek / Gemini / OpenAI）進行學術翻譯
  4. 三欄對照顯示（書影 / 原文 / 譯文），支持人工逐頁校改與重譯
  5. 導出 Word 對照文件，或保存 .json 校讀存檔以便隨時續校
鳴謝
本工具的 OCR 功能由 百度 PaddleOCR-VL 模型提供，通過 AI Studio 平台的公開 API 調用。感謝百度 AI 提供的高質量版面解析能力，使得複雜版式的古典文獻識別成為可能。
PaddleOCR 開源項目：github.com/PaddlePaddle/PaddleOCR
AI Studio API 申請：aistudio.baidu.com/paddleocr/task
